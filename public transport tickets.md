# Public Transport Ticket Gallery — Project Description

The public transport ticket gallery is a personal project Cristian built as a creative playground for design, motion, and front-end experimentation. The idea started from a simple habit of collecting public transport tickets from different cities, which eventually evolved into a curated digital archive where each ticket is displayed as a small piece of visual culture. Instead of treating it as a static gallery, Cristian used the project to explore interaction patterns, subtle animations, and a clean visual system that highlights the character of each ticket.

The website functions as a space to experiment freely with layout composition, color, typography, and motion in ways that don’t always fit into standard product design case studies. Each element is intentionally crafted, from the spacing and shadow rhythm to the hover states and transitions that bring the tickets to life. The project also acts as a front-end learning environment where Cristian could test ideas, refine micro-interactions, and push himself technically by integrating animation libraries and modern web tooling.

Beyond its visual and technical experimentation, the ticket gallery reflects Cristian’s interest in storytelling through simple artifacts. These small, mundane objects—bus, metro, and tram tickets—become a way to explore identity, travel memories, and design systems specific to different cities. The final product is a polished, public website supported by an open GitHub repository, showing not only Cristian’s eye for craft but also his ability to design, build, and ship a complete experience end-to-end.

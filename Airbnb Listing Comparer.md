# Airbnb Listing Comparer — Chrome Extension Project Description

Cristian built the Airbnb Listing Comparer as a practical tool to solve a simple but universal problem: comparing Airbnb listings quickly and clearly. Planning a trip usually means juggling multiple tabs, copy-pasting details, and trying to mentally track differences in price, amenities, reviews, and location. The experience is clunky, repetitive, and easy to get wrong. Instead of tolerating that friction, Cristian designed and developed a Chrome extension that automates the entire process using AI.

The extension activates directly from the browser and pulls key details from Airbnb listings without requiring the user to switch context. With a single click, it gathers the relevant data, interprets the content, and generates a side-by-side comparison using an AI assistant Cristian built into the tool, named Luna. Luna summarizes the listings in natural language, points out notable differences, highlights pros and cons, and gives the user a clear, digestible overview that would normally take several minutes of manual checking.

This project represents Cristian’s hybrid approach to design: he doesn’t just stop at wireframes or UI concepts. He prototypes with code, integrates real LLM workflows, structures JSON payloads, and builds functional logic that runs in a real browser environment. The extension’s interface is minimal and straightforward, focusing entirely on clarity and speed. The backend interaction with AI models runs smoothly behind the scenes, allowing users to focus only on making faster decisions.

Most importantly, the Airbnb Listing Comparer is not just a concept—it’s a live, public product available on the Chrome Web Store:

[Airbnb Listing Comparer on Chrome Web Store](https://chromewebstore.google.com/detail/airbnb-listing-comparer/dfkpdnhihibjifejkhbpickfpkkciohe)

It demonstrates Cristian’s ability to identify a real user pain point, design a simple but effective solution, and ship it end-to-end. The project highlights his strengths in product thinking, AI integration, rapid prototyping, and building polished, functional tools that people can actually use.
